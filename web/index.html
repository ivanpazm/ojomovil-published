Ojo Móvil

Convierte tu móvil en un ojo portátil que Alexa puede usar para hacer fotos cuando lo necesites.

Rápido, sencillo y útil.

⭐ ¿Qué es Ojo Móvil?

Una app que permite que Alexa use la cámara de tu móvil para hacer fotos bajo demanda.

Aprovecha cualquier Android que tengas por casa y úsalo como cámara remota sin complicaciones.

⭐ Qué puedes hacer

  • Vigilar tu casa o tu mascota.
  • Comprobar si todo está en orden cuando no estás.
  • Hacer fotos sin levantarte del sofá.
  • Darle una segunda vida a un móvil viejo.

⭐ Cómo funciona

  1. Instala la app en tu móvil Android.
  2. Vincúlalo con Alexa usando un código.
  3. Pide: Alexa, captura ahora.
  4. Foto hecha.

⭐ Servidor web incluido

Incluye un pequeño servidor web para ver la cámara desde cualquier navegador o Smart TV.

Muy práctico para usarlo desde el sofá.

Si lo dejas encendido mucho rato, gasta batería y puede funcionar como estufa de mano, cosa que aquí en Andalucía sobra la verdad.

⭐ Compatibilidad

  • Android 8+
  • Alexa con Skills
  • Cámara frontal o trasera
  • Funciona con WiFi normal
